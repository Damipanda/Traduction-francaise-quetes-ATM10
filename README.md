# Traduction-francaise-quetes-ATM10
Voici un humble projet, traduire le plus possible le modsPack ATM10 en francais 

Si vous voulez suivre l'avancement du projet voici un lien de la Roadmap: https://trello.com/b/JQNE887e/traduction-atm10

Installation: la structure du fichier est fait pour etre le plus facile vous devez donc deposer le fichier sous le chemin suivant : 
- disque principal(C en general)\user\votre nom\curseforge\minecraft\instance : et vous glisser deposer le fichier puis remplacer dans l'invite.
- si vous souhaiter contribuer au projet que ca soit pour traduire ou signaler des erreurs un lien de de contact sera ajouter bientot

en vous souhaitant un bon jeu a l'aide de cette traduction
